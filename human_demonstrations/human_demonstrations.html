Downloading the dataset

We recommend using the google-cloud-sdk to download our dataset as this handles interrupts and resuming downloads. You can download the demos as a single 1.8 gb gzipped tar bundle here or download with gsutil:
gsutil -m rsync gs://gibsonchallenge/behavior_human_demos_v0.4.0 ./behavior_human_demos_v0.4.0

We also offer a dataset which is useful for imitation learning approaches, including the human actions converted to the action space for our gym environment, as well as all observation modalities (RGB, depth, semantic segmentation, instance segmentation, task relevant object highlight) which is extremely large (300 gb). You can download this via gsutil:
gsutil -m rsync gs://gibsonchallenge/behavior_trajectories_v0.4.0 ./behavior_trajectories_v0.4.0
