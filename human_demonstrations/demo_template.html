Downloading the dataset

The human dataset is large (~150 gb with compressed hdf5s) and growing. We recommend using the google-cloud-sdk to download our dataset. We will be uploading additional demonstrations as they finish processing, up to a total of 500. Please report any issues.

The entire dataset processed for reinforcement learning (action, all observation modalities) can be downloaded with the following command:

gsutil -m rsync gs://gibsonchallenge/behavior_human_demos_v2 ./behavior_human_demos_v2

To download the raw hdf5s, which include eyetracking and individual pose of the VR sensors, please use the following command:

gsutil -m rsync gs://gibsonchallenge/behavior_human_demos_raw ./behavior_human_demos_raw
